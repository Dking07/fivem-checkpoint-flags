## DESCRIPTION
 
Custom race flag props for FiveM, built to mark checkpoints and start/finish lines on any race track or event setup.

The pack includes two prop types — **Checkpoint** and **Start/Finish** — each available in multiple color variations, ready to be spawned and used with any prop spawner or map builder.
 
## Props
 
### Checkpoint flags — `dking_checkpoint_*`
 
7 color variations · **1.4 MB each**
 
| Model name | Color |
|---|---|
| `dking_checkpoint_b` | Blue |
| `dking_checkpoint_g` | Green |
| `dking_checkpoint_p` | Pink |
| `dking_checkpoint_r` | Red |
| `dking_checkpoint_rd` | Red Dark |
| `dking_checkpoint_w` | White |
| `dking_checkpoint_y` | Yellow |
 
### Start / Finish flags — `dking_startfinish_*`
 
5 color variations · **13 KB each**
 
| Model name | Color |
|---|---|
| `dking_startfinish_g` | Green |
| `dking_startfinish_r` | Red |
| `dking_startfinish_rd` | Red Dark |
| `dking_startfinish_w` | White |
| `dking_startfinish_y` | Yellow |
 
## Features
 
- 12 props total — 7 checkpoint + 5 start/finish variations
- Multiple color options to match any race event aesthetic
- Lightweight start/finish props (17 KB) for minimal streaming overhead
- **Collision is only present on the largest stem** — the flag panels themselves have no collision, allowing props to be placed closely together or driven through without issues
